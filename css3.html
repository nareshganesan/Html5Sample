Introduction

February 13, 2014 by Naresh

CSS3 is a front end web technology. It is the current version of CSS which is 3. CSS3 is semantic meaning the tags used to present the content are meaningful. CSS3 is just used to display the content to the end user. Advantages of CSS3 over its predecessors are semantic, does not require heavy plugins to run video and audio files.

CSS3 is used to add style to present the data in HTML5.
