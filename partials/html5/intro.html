Introduction

February 13, 2014 by Naresh

Html5 is a front end web technology. It is the current version of html which is 5. Html5 is semantic meaning the tags used to present the content are meaningful. Html5 is just used to display the content to the end user. Advantages of Html5 over its predecessors are semantic, does not require heavy plugins to run video and audio files.

HTML5 is used to present or share content across internet.

Elements

HTML5 is made up of individual elements. These elements are like the barebone skeletal system of HTML5.

<!Doctype html>
<html>
    <head>
        <title>
        </title>
    </head>
    <body>
    </body>
</html>

Attributes

Attributes are the specific to indivvidual elements of HTML5. These attribute define the way the elements get displayed in the page. The following section highlights attribute as bold.

<!Doctype html>
<html>
    <head>
        <title>
        </title>
    </head>
    <body id="attribute1" class="atribute2">
    </body>
</html>
